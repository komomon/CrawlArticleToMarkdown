

## 文章爬取Bot😎使用介绍

> 防止文章没有看就删除了，所以借用大佬项目，自己用Electron写了个简单的应用，将公众号,知乎,知乎专栏,简书,知否(SegmentFault),掘金,CSDN,V2EX,博客园文章转为 markdown，封装成了应用。
